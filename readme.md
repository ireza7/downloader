# 📥 دانلودر فایل با ربات بله و گیتهاب

<div dir="rtl">

**فایل از هر لینکی دانلود کن، مستقیم توی ریپازیتوری گیتهاب ذخیره کن.**  
دو راه داری برای دانلود: ۱- ویرایش فایل `trigger_download.txt` ۲- ربات بله

> این پروژه اول برای تلگرام نوشته شده بود، فقط **اندپوینتش رو به بله تغییر دادیم.**  
> نام متغیر همون `TELEGRAM_TOKEN` مونده، ولی توکن بله رو توش می‌ذارید.

---

## 🚀 راه‌اندازی

### ۱. پروژه رو Fork کن

دکمه Fork رو بزن تا یه نسخه توی اکانت خودت داشته باشی.

### ۲. توکن بله رو بزار

از `BotFather` داخل بله یه ربات بساز و توکنش رو بگیر.

بعدش برو تو **Settings > Secrets and variables > Actions** و یه Secret جدید به اسم `TELEGRAM_TOKEN` بساز و توکن رو توش قرار بده.

### ۳. دسترسی‌ها رو تنظیم کن

تو همون بخش **Settings > Actions > General**:

- پایین صفحه، گزینه‌ی **Workflow permissions** رو روی **Read and write permissions** بذار.

### ۴. ورک‌فلو رو فعال کن

برو به تب **Actions**، اگه پیغام تأیید اومد قبولش کن.  
بعد روی `Telegram Bot (Hourly Long Poll)` کلیک کن و **Run workflow** رو بزن.

---

## 💬 استفاده از ربات بله

به رباتت پیام بده:

| دستور / پیام | کار |
|--------------|-----|
| یک لینک | همون فایل ساده ذخیره می‌شه |
| چند لینک | همه تو یه فایل zip |
| `/simple` + لینک‌ها | همه ساده ذخیره بشن |
| `/zipall` + لینک‌ها | همه تو یه zip |
| `/zipeach` + لینک‌ها | هر فایل جدا zip |
| `/list` | لیست فایل‌های ذخیره‌شده |

بعد دانلود، ربات لینک مستقیم فایل رو می‌فرسته.

---

## 📝 روش ویرایش فایل `trigger_download.txt`

فایل `trigger_download.txt` توی ریشه پروژه رو باز کن، لینک‌ها رو زیر هر کامنت که می‌خوای بذار.  
سه بخش داره:

- ساده
- همه تو یه zip
- هر کدوم تو یه zip جدا

بعدش Commit بزن. فایل‌ها خودکار دانلود و ذخیره می‌شن و فایل دوباره خالی می‌شه.

---

## 📁 فایل‌ها کجا ذخیره می‌شن؟

همه فایل‌های دانلود شده تو پوشه‌ی `downloads/` قرار می‌گیرن.

---

## 🛠 مشکلات رایج

- **ربات جواب نمی‌ده**: توکن درسته؟ `TELEGRAM_TOKEN` حتماً با توکن بله پر شده باشه. یک بار ورک‌فلو رو دستی اجرا کن.
- **خطای دسترسی**: مطمئن شو تو Actions تنظیمات Read & Write فعال باشه.
- **فایل‌ها نمی‌یان**: چند ثانیه صبر کن، صفحه‌ی گیتهاب رو رفرش کن.

---

## ✨ نکته‌ی پایانی

پروژه روی بله کار می‌کنه. فقط اندپوینت API عوض شده، همه چیز دیگه مثل قبله.  
حتی اسم متغیر `TELEGRAM_TOKEN` رو هم نگه داشتیم.
