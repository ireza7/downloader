# 📥 دانلودر فایل با ربات بله و گیتهاب

فایل از هر لینکی دانلود کن، مستقیم تو ریپازیتوری گیتهاب ذخیره کن.

دو راه داری:
۱. ویرایش فایل `trigger_download.txt`
۲. ربات بله (همان ربات تلگرام، فقط اندپوینت عوض شده)

---

## ✨ ویژگی‌ها

- نام واقعی فایل (از هدر یا URL نهایی گرفته می‌شود)
- دانلود تکه‌ای (Chunked) برای فایل‌های بزرگ
- سه حالت ذخیره: ساده، همه در یک zip، هر فایل در zip جداگانه
- اجرای سریع با کش pip و کانتینر سبک Alpine
- دریافت لیست فایل‌های ذخیره‌شده از طریق ربات

---

## 🚀 راه‌اندازی

۱. پروژه را **Fork** کن.

۲. توکن ربات بله را در **Settings → Secrets → Actions** با نام `TELEGRAM_TOKEN` اضافه کن.

۳. در **Settings → Actions → General**، گزینه‌ی **Workflow permissions** را روی **Read and write permissions** بگذار.

۴. به تب **Actions** برو، ورک‌فلو `Telegram Bot (5min Poll)` را انتخاب کن و یک‌بار **Run workflow** بزن.

---

## 💬 ربات بله

| دستور | کار |
|-------|-----|
| `/start` | خوش‌آمد گویی |
| `/help` | راهنما |
| `/list` | لیست فایل‌های ذخیره‌شده |
| `/simple` + لینک‌ها | دانلود ساده |
| `/zipall` + لینک‌ها | همه در یک zip |
| `/zipeach` + لینک‌ها | هر فایل در یک zip جدا |
| `/cancel` | لغو درخواست (همان چرخه) |
| ارسال لینک (بدون دستور) | اگر یک لینک باشد → ساده، اگر چندتا باشد → zipall |

پاسخ ربات حداکثر ۵ دقیقه طول می‌کشد.

## 📝 فایل `trigger_download.txt`

فایل سه بخش دارد. لینک‌ها را زیر کامنت مناسب بنویس و Commit کن.  
فایل‌ها خودکار دانلود و در `downloads/` ذخیره می‌شوند.

```
# اگر لینک های خود را در زیر این کامنت وارد کنید فایل ها بدون تغییر به صورت ساده ذخیره میشوند

# اگر لینک هارا زیر این کامنت وارد کنید همه فایل ها در یک فایل فشرده زیپ ذخیره میشوند

# اگر لینک هارا زیر این کامنت وارد کنید هر فایل به صورت یک فایل فشرده زیپ ذخیره میشوند
```

## 🛠 رفع مشکلات

- **ربات پاسخ نمی‌دهد**: توکن را چک کن، یک‌بار Workflow را دستی اجرا کن.
- **فایل‌ها ظاهر نمی‌شوند**: چند ثانیه صبر کن و صفحه را رفرش کن.
- **خطای ۴۰۳**: دسترسی Read & Write در Actions فعال باشد.
